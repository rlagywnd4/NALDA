# 🪽NALDA
NALDA - "날아다니다"와 "배달"의 의미를 융합해 빠르고 경쾌한 이미지를 전달.

## ✨프로젝트 소개
배달의 민족을 모티브로 하여 로그인을 2가지로 나누어서 배달을 시키는 고객과 배달을 해주는 가게 2개로 나누어서 관리하고 있습니다.

## 📆프로젝트 기간
2025.01.07 ~ 2025.01.13
